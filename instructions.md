# Instruções para utilizar no Gitpod via terminal:

## Acessar MySQL:
```
mysql
```

## Exibir todos os bancos de dados:
```
show databases;
```

## Acessar o banco de dados:
```
use <Nome_database>;
```

## Ver todas as tabelas:
```
show tables;
```

## Ver os dados de uma tabela:
```
select * from <Nome_tabela>
```
